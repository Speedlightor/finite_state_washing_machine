# Finite State Washing Machine (VHDL)

RTL-модель стиральной машины на основе конечного автомата (автомат Мура), реализованная на языке VHDL. Проект включает самопроверяемые testbench, граф переходов конечного автомата, временные диаграммы, полученные при симуляции, а также версию, портированную на отладочную плату **Sipeed Tang Primer 20K Dock**.

## Возможности модели

- Полный цикл стирки: набор воды → стирка → слив → полоскание → отжим → индикация завершения;
- защита от запуска при открытой двери;
- программная защита кнопки «Старт» от дребезга (кнопка должна удерживаться 0,25 с);
- датчик уровня воды: этап набора завершается по факту заполнения бака, а не по таймеру;
- состояние ошибки `S_ERROR` с отдельным светодиодом: срабатывает по таймауту набора воды (кран перекрыт) и при незавершённом сливе (засор);
- отсчёт длительности этапов в секундах через делитель тактовой частоты;
- параметр `generic CLK_FREQ`, позволяющий ускорять «секунду» в симуляции без изменения кода для платы.

## Конечный автомат

| Состояние | Этап | Активные исполнительные сигналы | Длительность |
|-----------|------|--------------------------------|--------------|
| `S_IDLE`  | ожидание запуска | — | до нажатия «Старт» при закрытой двери |
| `S_FILL`  | набор воды | `water_valve` | до срабатывания датчика уровня (таймаут 8 с → `S_ERROR`) |
| `S_WASH`  | стирка | `motor_on` | 20 с |
| `S_DRAIN` | слив мыльной воды | `drain_pump` | 6 с (если вода не слилась → `S_ERROR`) |
| `S_RINSE` | полоскание | `water_valve`, `motor_on` | 12 с |
| `S_SPIN`  | отжим | `motor_on`, `motor_fast`, `drain_pump` | 10 с |
| `S_DONE`  | завершение | `done_led` | до открытия двери |
| `S_ERROR` | ошибка | `error_led` | до аварийного сброса `reset` |

Граф переходов и временные диаграммы находятся в соответствующих папках проекта.

## visual_washing_machine — ранняя версия

Учебная модель без привязки к конкретному устройству. Активные уровни всех сигналов прямые (`'1'` = активен), длительности этапов заданы в тактах, что делает временные диаграммы компактными и наглядными. Подходит для изучения принципов построения конечных автоматов и self-checking testbench на VHDL.

## device_realiz_washing_machine — версия для Tang Primer 20K Dock

Версия, доведённая до работы на реальной плате:

- тактирование от бортового генератора **27 МГц** (`generic CLK_FREQ = 27_000_000`);
- секундный делитель частоты: длительности этапов заданы в секундах;
- антидребезг кнопки «Старт»: нажатие засчитывается после удержания 0,25 с;
- кнопки `reset` и `start` **активны низким уровнем** (`'0'` = нажата);
- светодиоды `done_led`, `error_led` и шина `state_out` **зажигаются низким уровнем**, поэтому код состояния выводится инвертированным.

### Назначение выводов (фрагмент .cst)

| Сигнал | Ножка | Назначение |
|--------|-------|-----------|
| `clk` | H11 | бортовой генератор 27 МГц |
| `reset` | T3 | кнопка аварийного сброса |
| `start` | C7 | кнопка «Старт» |
| `door_closed` | E9 | переключатель «дверь закрыта» |
| `water_full` | E8 | переключатель «бак заполнен» |
| `state_out[2:0]` | L16, L14, N14 | светодиоды кода состояния |
| `done_led`, `error_led` | внешний модуль | светодиоды завершения и ошибки |

Полный файл ограничений находится в папке проекта.
